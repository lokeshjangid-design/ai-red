üìã FIRST TIME SETUP (One Time Only)
------------------------------------
1. Double-click: SETUP.bat

   - Installs all Node.js packages

üöÄ STARTING THE SYSTEM (Every Time)
------------------------------------
1. Double-click: START_SYSTEM.bat
   - Opens 2 command windows
   - Window 1: Backend Server (Flask)
   - Window 2: Frontend Dashboard (React)

2. Wait 10-15 seconds for servers to start

3. Browser will auto-open: http://localhost:3000
   (If not, manually open this URL)


üì§ USING THE DASHBOARD
-----------------------
1. Click "Upload Traffic Video" area
2. Select your video file (MP4, AVI, MOV, MKV)
3. Choose number of lanes (2-6)
4. Click "Analyze Video" button
5. Wait for processing (progress shown)
6. View results:
   - Total vehicle count
   - Lane-wise counts (L1, L2, L3, L4...)
   - Vehicle types (cars, bikes, trucks, buses)
   - Processed video with detection boxes

üîß TROUBLESHOOTING
------------------
Problem: "pip not found"
Solution: Install Python 3.8+ from python.org

Problem: "npm not found"
Solution: Install Node.js 16+ from nodejs.org

Problem: Backend won't start
Solution: Check if Python is installed: python --version

Problem: Frontend won't start
Solution: Check if Node is installed: node --version

Problem: Video processing fails
Solution: Ensure video file is valid MP4/AVI/MOV/MKV

Problem: Out of memory
Solution: Use smaller video or reduce quality
